# 📊 SQL Daily Practice

Welcome to my **SQL Daily Practice** repository!

This repository documents my journey of solving SQL interview questions daily to strengthen my problem-solving skills and prepare for Data Analyst/Data Scientist interviews.

---

## 🎯 Goals

- Solve SQL problems consistently.
- Learn different SQL concepts and patterns.
- Improve query optimization and readability.
- Build a reference of commonly asked interview questions.
- Track my learning progress over time.

---

## 📚 Topics Covered

- SELECT Statements
- Filtering (`WHERE`)
- Sorting (`ORDER BY`)
- Aggregations (`COUNT`, `SUM`, `AVG`, `MIN`, `MAX`)
- `GROUP BY` & `HAVING`
- Joins
  - INNER JOIN
  - LEFT JOIN
  - RIGHT JOIN
  - SELF JOIN
- Subqueries
- Common Table Expressions (CTEs)
- Window Functions
- CASE Statements
- Date & Time Functions
- String Functions
- Ranking Functions
- Conditional Aggregation
- Interview SQL Patterns
